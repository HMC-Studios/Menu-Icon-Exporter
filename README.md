# Menu Icon Exporter for Blockbench

A Blockbench plugin that exports your models as high-quality icons with full camera control.

![Menu Icon Exporter Preview](images/preview.png)

## Features

- Auto-frames your model perfectly
- Manual camera controls (zoom, pan, rotate on all axes)
- Multiple export sizes (16x16 to 128x128 or custom)
- Live preview with real-time updates
- Quick export buttons for common sizes
- Keyboard shortcut (Ctrl+Shift+I)

## Installation

### From Blockbench (Recommended)
1. Open Blockbench
2. Go to File → Plugins
3. Search for "Menu Icon Exporter"
4. Click Install

### Manual Installation
1. Download `menu_icon_exporter.js`
2. In Blockbench, go to File → Plugins → Load Plugin from File
3. Select the downloaded file

## Usage

- **Main Export**: File → Export → Export Menu Icon (Ctrl+Shift+I)
- **Quick 16x16**: File → Export → Quick Export 16×16 Icon
- **Quick 64x64**: File → Export → Quick Export 64×64 Icon

## Requirements

- Blockbench 3.6 or newer
- Works with all model formats
